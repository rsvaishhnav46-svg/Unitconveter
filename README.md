# Unit Converter

A simple web-based unit converter for length units.

## Usage

Open `index.html` in your web browser.

Enter a value, select the "from" unit and "to" unit, then click the "Convert" button.

The result will be displayed below.

## Supported Units

- Meters (m)
- Feet (ft)
- Inches (in)

## Features

- Converts between meters, feet, and inches.
- Input validation for numeric values.
- Clean, responsive UI.

## No Dependencies

This project uses only vanilla HTML, CSS, and JavaScript. No external libraries or frameworks are required.
